## 链路

```
Step 1                    Step 2                    Step 3
造角色                    配声音                    开口说话
┌───────────────────┐    ┌───────────────────┐    ┌───────────────────┐
│ character-library │    │ voice-design-     │    │ character-talking │
│ -generator        │───→│ pipeline          │───→│ -video-prompt-    │
│                   │    │                   │    │ builder           │
│ 366 原型          │    │ 图片→VLM→音色     │    │ Voice DNA + 情绪  │
│ × 14 维 Face DNA  │    │ →台词→试听        │    │ + 方言→视频 prompt│
```
